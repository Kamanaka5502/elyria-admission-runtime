# Pricing Structure

## Consequence Twin Scan

Recommended range: **$7,500-$15,000**

Use the lower end only for a narrow scope with limited consequence pathways.

Use the higher end when the scope includes automation, payment movement, access control, deployment decisions, multiple systems, sensitive operations, or executive-ready reporting.

## Implementation Pilot

Recommended range: **$25,000-$75,000**

Pilot pricing depends on:

- number of governed paths
- system integration depth
- evidence and receipt requirements
- replay requirements
- stakeholder count
- operational sensitivity
- implementation timeline
- client material quality

## Runtime Layer

Scoped only after pilot validation.

Do not price runtime before confirming:

- consequence-node inventory
- authority model
- evidence sources
- custody boundaries
- refusal requirements
- receipt format
- replay requirements
- integration constraints
